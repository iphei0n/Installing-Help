  ## How to Install RVC
  Visit https://github.com/dr87/Vonovox.
  Click the [Code] button and select Download ZIP.
  Extract the files, run setup.bat, and then run start.bat.
  Upload your downloaded model files to the Model Presets section.
  
  ## Virtual Audio Cable (VAC) Installation
  Download the Lite version: vac470lite.zip.
  Install the program and reboot your computer.
</details>


<details>
  <summary>日本語ガイドを表示</summary>
  ## RVCのインストール方法
  https://github.com/dr87/Vonovox にアクセスします。
  [Code] ボタンをクリックし、Download ZIP を選択してダウンロードします。
  展開したフォルダ内の setup.bat を実行し、その後に start.bat を実行します。
  ダウンロードしたモデルファイルを Model Presets にアップロードします。
  
  ## 仮想オーディオケーブル (VAC) のインストール
  Lite版をダウンロード: vac470lite.zip
  インストール後、必ず 再起動 してください。
</details>
